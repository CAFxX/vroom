## Executive summary
* Very high end RISC-V implementation – cloud server class
* Out of order, super scalar, speculative
* RV64-IMAFDCHB
* Up to 8 IPC (instructions per clock) peak  
* 2-way simultaneous multithreading capable
* Multi-core
* GPL3 – dual licensing possible
